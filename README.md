# Simple Weather App

A lightweight, responsive web application that displays current weather conditions for any city. Built using vanilla JavaScript, HTML, and CSS.

# About The Project

This project is a simple Weather Dashboard designed to practice DOM manipulation and API integration using JavaScript. It allows users to search for a specific location and instantly view weather details such as temperature, humidity, and wind speed.

This application is client-side only and does not require a backend database.

# Features

Real-time Weather Data: Fetches up-to-date data (via API).

Search Functionality: Search for weather by city name.

Responsive Design: Works on desktop and mobile devices.

Dynamic Backgrounds: (Optional) Background changes based on weather conditions.

No Database: Lightweight and fast.

Language: JavaScript (ES6+)

Frontend: HTML5, CSS3

Editor: Notepad++ (or any standard text editor like VS Code, Sublime Text)

Database: None (Stateless)
